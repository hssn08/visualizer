  <behavior>
    layout tests:
    - getLayoutedElements returns nodes with updated x,y positions (not all at 0,0)
    - getLayoutedElements with direction='TB' sets targetPosition='top' and sourcePosition='bottom' on nodes
    - getLayoutedElements with direction='LR' sets targetPosition='left' and sourcePosition='right' on nodes
    - Returned nodes have same count as input nodes
    - Returned edges are unchanged (same reference or same content)
    - Handles empty nodes array without error
    - Handles single node without error
    - Handles nodes with no edges (disconnected graph) without error
    - Node positions are offset by half NODE_WIDTH/NODE_HEIGHT from dagre center (not raw dagre center values)
  </behavior>
  <action>
    1. Run `npm install @dagrejs/dagre` to add the dependency.

    2. Create src/lib/layout.ts:
       - Import dagre from '@dagrejs/dagre'
       - Import type { Node, Edge } from '@xyflow/react'
       - Export constants: NODE_WIDTH = 250, NODE_HEIGHT = 150 (generous for badge+handle content)
       - Export function getLayoutedElements(nodes: Node[], edges: Edge[], direction: 'TB' | 'LR' = 'TB'): { nodes: Node[]; edges: Edge[] }
         - Create new dagre.graphlib.Graph(), setDefaultEdgeLabel(() => ({}))
         - Set graph options: rankdir=direction, nodesep=80, ranksep=100
         - Add each node: g.setNode(node.id, { width: NODE_WIDTH, height: NODE_HEIGHT })
         - Add each edge: g.setEdge(edge.source, edge.target)
         - Run dagre.layout(g)
         - Map nodes to new positions:
           - const pos = g.node(node.id)
           - targetPosition: direction === 'LR' ? 'left' : 'top'
           - sourcePosition: direction === 'LR' ? 'right' : 'bottom'
           - position: { x: pos.x - NODE_WIDTH / 2, y: pos.y - NODE_HEIGHT / 2 }
         - Return { nodes: layoutedNodes, edges } (edges pass through unchanged)

    3. Write tests verifying position computation, direction handling, edge cases.
