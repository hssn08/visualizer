Create `src/hooks/useNodeDelete.ts`:
- Export `useNodeDelete()` hook that encapsulates the onBeforeDelete Promise pattern from RESEARCH.md Pattern 4
- Manages `deleteConfirm` state: `{ nodes: Node[]; edges: Edge[]; resolve: (value: boolean) => void } | null`
- Returns `{ onBeforeDelete, deleteConfirm, confirmDelete, cancelDelete }`:
  - `onBeforeDelete({ nodes, edges })`: If `nodes.length === 0`, returns `true` immediately. Otherwise creates a new Promise, stores resolve in state, returns the Promise.
  - `confirmDelete()`: calls `deleteConfirm.resolve(true)`, clears state
  - `cancelDelete()`: calls `deleteConfirm.resolve(false)`, clears state

- Test: calling `onBeforeDelete({ nodes: [], edges: [...] })` resolves to true immediately (no dialog state set)

- Test: calling cancelDelete() resolves the Promise to false and clears state
- Test: deleteConfirm state contains the nodes and edges passed to onBeforeDelete
