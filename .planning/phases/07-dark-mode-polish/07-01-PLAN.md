<objective>
Add dark mode support and collapsible side panels.

Purpose: UI-01 (dark mode via shadcn theme) and UI-05 (collapsible property panel and node palette) are the visual polish requirements for the final phase. Dark mode is 90% in place via existing CSS variables; this plan wires the toggle and propagates theme to the two subsystems that bypass CSS variables (React Flow canvas elements and json-edit-react). Collapsible panels let users maximize canvas space.

Output: Working dark/light/system theme toggle in toolbar, all components dark-mode-aware, collapsible palette and property panel with toggle buttons.
</objective>
